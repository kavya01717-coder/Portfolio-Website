# Kavya Patel — Portfolio Website

> 🔗 Live: [kavya-patel17.netlify.app](https://kavya-patel17.netlify.app)

Personal portfolio built with pure HTML, CSS & JavaScript — no frameworks, no dependencies.

## Features
- 🌗 Dark / Light mode
- 🖱️ Custom cursor & scroll animations
- 📱 Fully responsive
- 📁 Resume modal (View & Download)
- 📋 Copy to clipboard for email & phone

## Tech Stack
`HTML5` `CSS3` `JavaScript` `Netlify`

## Run Locally
```bash
git clone https://github.com/kavya01717-coder/portfolio.git
cd portfolio
open index.html
```
